### About
The dead internet theory is a belief that the majority of internet traffic is coming from bots and AI generated content.

### Dataset
The original dataset can be found on [Kaggle](https://www.kaggle.com/datasets/nudratabbas/the-dead-internet-theory-reddit-bot-vs-human).
It contains the data of 500 comments across 11 different features. Notably, this includes a unique ID for each comment.
The target is the boolean is_bot_flag variable, showing whether or not the comment was generated by a bot.

### Key features:
Key features picked include:
- Account age (days)
- Reply delay (seconds)
- Average word length
- Sentiment score
- User karma
- Presence of links
